# Design Reference Assets

This folder stores local website reference captures for the Saint Laurent Valentine Gift Campaign demo.

## Reference Sources

### `brunello-cucinelli-ai/`

Use this package as the workflow and interaction reference.

Open `references/brunello-cucinelli-ai.html` to inspect the saved page with local assets.

- Reference the AI shopping assistant entry point.
- Reference guided discovery and recommendation UX.
- Reference prompt-driven product exploration.
- Do not copy Brunello Cucinelli visual identity, typography, product styling, color palette, or brand tone.

中文说明：

- `brunello-cucinelli-ai/` 用作 UI/UX workflow 参考。
- 重点参考 AI 导购入口、推荐流程、用户如何通过 prompt 探索商品。
- 不复制 Brunello Cucinelli 的品牌视觉、字体、商品风格、色彩或审美调性。

### `saint-laurent-cn/`

Use this package as the brand and commerce reference.

Open `references/saint-laurent-cn.html` to inspect the saved page with local assets.

- Reference Saint Laurent China official website typography, header, footer, spacing, image treatment, and visual tone.
- Reference Saint Laurent product imagery and fashion commerce categories.
- The demo should use Saint Laurent's luxury-minimal aesthetic.
- Product recommendations should focus on bags, accessories, ready-to-wear, and shoes.

中文说明：

- `saint-laurent-cn/` 用作 Saint Laurent 官网品牌和电商参考。
- 重点参考字体、header、footer、间距、图片处理方式和整体审美调性。
- Demo 应保持 Saint Laurent 的极简奢侈品气质。
- 推荐商品聚焦包袋、配饰、成衣和鞋履。

## Implementation Rule

BC defines the interaction logic. YSL defines the brand expression.

中文：

BC 定义交互逻辑；YSL 定义品牌表达。
